# CollabFlow

Run the backend server (from workspace root):

```bash
npm run start        # runs backend/src/server.js
npm run dev          # runs backend using nodemon via npx
```

Or run directly from the backend folder:

```bash
cd backend
npm install
npm start            # starts the server
npm run dev          # requires nodemon in backend devDependencies
```

VS Code

- Use the `Launch Backend` or `Launch Backend (Nodemon via npx)` configurations in the Debug panel.
